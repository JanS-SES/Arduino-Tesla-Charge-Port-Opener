# Data Samples

This folder contains the original capture and conversion material used to create the bitstream in the Arduino sketches. The main time-domain sample was recorded with a LabNation SmartScope logic analyzer.

## Files

- `Bitstream & Array.txt` - raw bitstream plus the Arduino array representation used by the sketches.
- `Sample to Bitstream conversion.xls` - spreadsheet used during manual conversion.
- `Scopesample/` - LabNation SmartScope logic analyzer captures and screenshots.
- `RF Sample/` - SDR-derived RF samples, including an `rtl_433` OOK export.

These files are intentionally kept in the repository for traceability. The sketches do not need them at runtime.
